🎯 Elemen Diagram

👤 Actor (Aktor)
Pengguna sistem (Konsumen, Supir, Admin)

🔄 Aktivasi (Activation Box)
Periode partisipan sedang aktif (garis vertikal)

📨 Message
Komunikasi antar partisipan (panah solid/putus)

✏️ Tipe Interaksi

→ Solid Arrow
Synchronous call (menunggu respon)

⇢ Dashed Arrow
Response/Return (balasan dari call)

+ Activation
participant->>+ = start, -->>- = end

🔲 Kontrol Flow

alt ... else
Conditional logic (jika/maka/atau)

opt
Optional flow (opsional, mungkin terjadi)

rect / note
Pengelompokan steps dengan label

💡 Tips Membaca: Diagram dibaca dari atas ke bawah. Garis vertikal menunjukkan waktu yang berlalu. Setiap panah adalah satu interaksi/message antar partisipan.
